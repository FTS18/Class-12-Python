## Run Locally
Clone the project
```bash
git clone https://github.com/FTS18/Python-Projects/
```
Install Libraries
```bash
pip install -r requirements.txt
```
Go to the personal/school directory
```bash
  cd personal
```
To run this project type the following command
```bash
  python f"{filename}.py
```
